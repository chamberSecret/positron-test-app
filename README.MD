# Positron Test App
___

## Как начать?

1. Склонировать 
2. В корне проекта прописать `make`
3. Прописать `yes`
4. Подождать
5. Открыть браузер и перейти на localhost:8080
6. Для входа в админ панель: admin@admin.ru / admin
